# Vrinda-Store-Annual-Sales-Dashboard-Excel-
Developed an interactive Excel dashboard to analyze Vrinda Store's annual sales performance using Pivot Tables, Pivot Charts, Slicers, and KPIs.

Project Overview:
Developed an interactive Excel dashboard to analyze Vrinda Store's annual sales performance using Pivot Tables, Pivot Charts, Slicers, and KPIs.

Key Insights
Women contributed 64% of total sales.
Customers aged 30–49 years generated the highest revenue.
Maharashtra, Karnataka, and Uttar Pradesh were the top-performing states.
Amazon (35%), Myntra (23%), and Flipkart were the leading sales channels.
92% of orders were successfully delivered.
Business Conclusion
Focus marketing campaigns on women aged 30–49 years.
Increase promotions in Maharashtra, Karnataka, and Uttar Pradesh.
Prioritize advertising and offers on Amazon, Myntra, and Flipkart to maximize sales.

Tools: MS Excel, Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, Dashboard
